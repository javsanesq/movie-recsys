# Security Policy

movie-recsys is a local, single-reviewer demo trained on the public MovieLens
25M dataset. It has **no authentication** and is not intended for public
deployment as-is.

- No secrets, credentials, or personal data are stored in this repository.
- All model artifacts are derived from the public MovieLens 25M dataset.
- The API exposes read-only recommendation/metrics endpoints over local data.

## Reporting

Found an issue? Open a GitHub issue on this repository. There is no private
disclosure process — this is a portfolio project, not a production service.
